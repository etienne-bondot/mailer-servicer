# Mailer

## What for

A mailing service.

## Required

You need to provide a `.env` file in the root directory with following credentials:

```text
GMAIL_USERNAME=
OAUTH_CLIENT_ID=
OAUTH_CLIENT_SECRET=
OAUTH_REFRESH_TOKEN=
OAUTH_ACCESS_TOKEN=
```

## Installation

`yarn`

## Start

`yarn start`

## Build

`yarn build`

## Test

`yarn test`

## Implementation

You can find a implementation example in `script/test-template.script`.
